Making a Space Shooter using Canvas

Dec 3, 2017

The goal of this tutorial is to create a basic space shooter using HTML5 from the ground up. Along the way, we will learn concepts such as Javascript prototypes and Canvas animation. While we will start initially with the keyboard, we will look at how to make this work on a mobile device.

Let’s start by creating a triangle, which will be our ship. We will use a separate canvas object for each sprite.
