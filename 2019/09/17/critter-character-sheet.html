SVG Character Sheet: Fuzzy Alien Critter

Sep 17, 2019

Intro

Rediscovering SVG’s has been an absolute joy because of their responsiveness, their editability in text editors, and their ability to look sharp no matter how they are scaled. It occurred to me that I want to step away from circular emoji faces as I embark on my next round of SVG animation posts, so I developed a character sheet for this fuzzy critter over the weekend for an upcoming post on animating an SVG path’s d attribute.

It would be all too easy to make a few terrifying moods here, but not today.

Materials

  • Inkscape, a free open-source vector image editor
  • SVG OMG (Jake Archibald), a free online image optimization tool
  • Sublime Text 2 for the coding

I actually did not use my graphics tablet for this one — the Inkscape tools gave me the control I needed.

Inkscape Process

An analogous process can be done using another vector image authoring tool, such as Adobe Illustrator or Sketch.

  • Created an initial character in Inkscape using the freehand draw, bezier, and ellipse tools
  • Copy and pasted the character multiple within the file
  • Used the “align” tool in Inkscape to get the eyes right
  • Modified the path with the bezier and draw tools
  • For the simpler mouths, removed nodes here and there
  • Added my signature with the “ellipse” style

Tech Process

  • Ran the image through SVG OMG to reduce unnecessary markup
  • Downloaded and saved the image in the subdirectory within my site.
  • Used the img element to pull it into the webpage.
  • In the SVG file, I adjusted the viewbox attribute of the SVG ever so slightly to make the image more centered.

Potential Uses

  • With some fine-tuning, an SVG like this could be used as a sprite sheet for animation.
  • If you separate the character states into separate smaller SVG files.
  • You can then inline one of the character states by pasting the markup from a file directly in your webpage. Then you could animate certain parts of it, such as the mouth and eyes, with CSS or JS.
  • You could include a background, either as a separate image, a separate element with a background CSS image, or as an additional element within the character SVG itself.
  • You could include an SVG as an object element and use a PNG fallback.

I will plan to write about some or all of these techniques in some upcoming posts. I am especially looking forward to animating the mouth by changing the d-attribute of the mouth path with JS. More to come!
