Optimizing an SVG Image with SVG OMG

Sep 3, 2019

Background

Graphics Programs like Inkscape and Illustrator are terrific for creating SVG images, but the resulting code can be bloated with a lot of unnecessary code such as metadata. Optimizing your SVG can make it enormously more readable and reduce its size by several boat loads, but getting your optimization settings just right can be tricky.

Fortunately, the free online tool SVG OMG has you covered if your export settings aren’t doing the whole job. In this post, we’ll look at SVG OMG and compare some code snippets from the “before” image and the “after” image.

The inkscape file was 20.14K and had a lot of unnecessary data in it. Immediately I wanted to optimize the file, so I remembered the SVG OMG tool from a podcast by SVG Expert Sarah Soueidan.

A little background

SVG OMG is an application created by Jake Archibald that is a GUI (Graphic User Interface) version of SVGO, an SVG optimization tool in node.js. You can download an offline version from his Github and

I uploaded my file. While keeping the default settings, I found that SVG OMG reduced the file size from 20.14k to 4.22k — 20.93% of its original size.

NOTE: Finder shows that the file size is larger.

Upon opening the file in Sublime Text, the extra unnecessary data was gone. The code looks readable, as if someone could have drawn these lines in code.

Here is the original file and the optimized file for reference.

One thing I see is that all the circles are grouped together, so I will be separating these faces into different SVG objects once we start taking a closer look at the code under the hood.

Optimizing SVG's can remove Inkscape layers. Makes perfect sense but I didn't automatically think of it. In my case I didn't mind, but this may cause issues if you want to go back and edit your image. So hold on to those original files in case you want to go back to your layers.

The dragon

After that, I tried it on the dragon image. THis reduced the size of the SVG by 50%.

Takeaways

  • SVG OMG is a terrific tool for optimizing images created with a graphics editor.
  • There is an online version and an offline version.
  • There are tons of settings that I recommend playing around with.
